# **App Name**: Numeric Echo

## Core Features:

- Basic Arithmetic: Perform basic arithmetic operations: addition, subtraction, multiplication, and division.
- Textual Conversion: Convert numerical expressions into their textual equivalents (e.g., '2+2' becomes 'two plus two').
- Answer Conversion: Convert the numerical result into its textual equivalent (e.g., '4' becomes 'four').
- Combined Display: Display both the numerical expression/result and their textual equivalents below.
- Initial Zero Display: The number input always shows '0' on initial load.
- Non-Editable Text Display: A non-editable text display box always shows 'zero' on initial load and provides the equivalent textual representation of the numerical operations performed. A tool that uses an LLM to perform textual representations.
- Off Button: A button to clear both the number and text displays.

## Style Guidelines:

- Primary color: Soft pink (#F4B4C4) for the body, creating a gentle and inviting feel.
- Background color: White (#FFFFFF) for the calculator form to provide a clean, simple layout.
- Accent color: Dark grey (#4A4A4A) for button text to ensure legibility.
- Font: 'PT Sans' for a humanist sans-serif style in both numbers and equivalent words.
- 3D effect buttons in light grey for a tactile, user-friendly interface.
- Clear separation between numerical input, textual display, and buttons for easy readability and use.
