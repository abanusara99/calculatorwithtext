# Calculator With Text (Speech)

This is a responsive, feature-rich calculator application built with Next.js, React, and Tailwind CSS. It provides standard calculation functionalities with a unique twist: it translates the numbers and mathematical expressions into words in real-time.

For detailed technical information, component breakdowns, and troubleshooting guides, please see [DOCUMENTATION.md](./DOCUMENTATION.md).

**Why create Calculator with text ?**

Calculator with text for kids and people who have hard time in reading the numbers in text format. My personal experience, I mostly read upto thousand (1000) , I confuse inbetween international/ worldwide system and indian system and I failed to pronounce. I wanted kids and people to read and use calculator for both calculation and pronunciation. I hope you will like and support me.

## Features

- **Standard Arithmetic Operations**: Perform addition, subtraction, multiplication, division, and modulus.
- **Real-time Text Conversion**: As you type your expression, it's instantly converted into words.
- **Text-to-Speech**: Click the speaker button (ðŸ”Š) to have expressions and results read aloud. This feature uses the browser's native **Web Speech API**, so it works offline and is completely free. A clear, high-quality voice is used to help with understanding and pronunciation.
- **Dual Numbering Systems**: Switch between:
  - **International System (WWS)**: Millions, Billions, etc.
  - **Indian System (IS)**: Lakhs, Crores, etc.
- **Keyboard and Button Input**: Use your keyboard for quick input or click the on-screen buttons.
- **Editable Display**: The numeric input field allows for full cursor control, editing, and correction.
- **Responsive Design**: A clean, modern interface that works seamlessly on desktop and mobile devices.

## Getting Started

To run this project locally:

1.  **Clone the repository.**
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
